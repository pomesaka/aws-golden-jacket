Topic: EFS for shared POSIX file system.

Key note:
Multiple EC2 instances need the same POSIX file system with minimal app change and managed service. EFS is the match.

Learning note:
EBS is block storage attached to EC2. EFS is a managed network file system for shared access from multiple instances.
