Use primary AWS sources first. Secondary blogs can be useful for tactics, but the source of truth should remain AWS official documentation and exam pages.

## Rule

When a claim affects exam order, Golden Jacket eligibility, retirement dates, or service behavior, verify it against AWS official sources before treating it as stable.
