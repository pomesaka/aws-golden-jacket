# AWS Certified Solutions Architect - Associate

Exam code: SAA-C03

Purpose: first calibration exam for the Golden Jacket roadmap.

## Official exam facts

- Category: Associate
- Duration: 130 minutes
- Format: 65 questions, multiple choice or multiple response
- Scored questions: 50
- Unscored questions: 15
- Passing score: 720 on a 100-1,000 scaled score
- Testing options: Pearson VUE test center or online proctored exam

## Why this exam first

SAA validates the exact skill that showed up in the initial diagnostic: requirement-based architecture reasoning.

It also exposes AWS-specific vocabulary gaps before moving to Professional and Specialty exams.
