# SAA Exam Realism Note

Current practice questions are slightly obvious on purpose because they target weak areas.

Real exam questions are usually more subtle.

Expected differences:

- more similar-looking services in the same question
- longer scenarios
- stronger cost or operational constraints
- multiple response questions
- fewer obviously bad options
- answers that are technically possible but not most appropriate

Practice mode plan:

- Q11 to Q15: reinforce weak areas with clearer signals.
- After that: increase realism by making distractors closer.
- Before booking: do timed answer-only practice.

Grading rule:

If the answer is correct but the reason is shallow, record it as correct but not fully diagnostic.
